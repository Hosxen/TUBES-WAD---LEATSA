1. buka cmd
2. cek composer dan laravel
3. bikin projek laravel
4. buka projek laravel di vscode
5. jalankan projek
6. buka routes-web (jelaskan) dan buka views (jelaskan)
7. bikin file index di views dan coba jalankan routes di file baru
8. bikin file jadwal di views dan coba jalankan
9. buat route controller 'root' lalu buat controller diterminal (php artisan make:controller PagesController)
10. buat route controller 'jadwal' lalu buat controller diterminal (php artisan make:controller JadwalController --resource)
11. coba jalankan
12. buat database di phpmyadmin
13. konfigurasi koneksi database (.env) (config-database)
14. buat tabel
15. buat migrasi lagi (php artisan make:migration create_schedules_table)
16. buka migrasi yg sudah dibuat, isi fieldnya
	varchar = string
	char = char
	unique = gaboleh sama
17. bikin (php artisan migrate) (ada users dan passwd untuk melakukan login dan registrasi dgn mudah)
18. bikin model untuk masukan data di migrate ke tabel jadwal dgn cara laravel
    (php artisan make:model schedules) ada di app-models
    panggil model di controller dgn cara $jadwal = \app\models\schedules
19. seeder
